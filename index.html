Documentation

Design Process

The goal of this project was to explore how workout duration, workout type, and heart rate influence calories burned. I began by identifying key variables in the dataset and experimented with different layouts before deciding on a dashboard-style visualization. The design places filters at the top and charts below, allowing users to interact with the data easily.

Initially, I considered using a single chart, but I realized that combining a scatterplot and bar chart would provide better insight. This approach allows users to both analyze relationships and compare categories.

Design Rationale

A scatterplot was chosen because it effectively shows the relationship between workout duration and calories burned. This helps users quickly identify trends and patterns. A bar chart was added to compare average calories burned across workout types, making differences between categories more visible.

Color was used to distinguish workout types, and interactive filters were added to allow users to explore subsets of the data. Tooltips were implemented so users can view detailed information when hovering over points.

Insights & Findings

The visualization reveals a clear positive relationship between workout duration and calories burned. Longer sessions tend to result in higher calorie expenditure. Additionally, higher heart rates are associated with increased calorie burn, suggesting that workout intensity plays an important role.

Different workout types show varying levels of efficiency in calorie burning. By using filters, users can explore how these patterns change across gender and age groups, providing deeper insight into workout performance.
